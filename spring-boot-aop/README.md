# [探秘Spring AOP](https://www.imooc.com/learn/869)
简介：面向切面编程（AOP）提供另外一种角度来思考程序结构，通过这种方式弥补了面向对象编程（OOP）的不足。同时AOP框架也是Spring的一个关键的组件在平时的面试中也是一个容易被询问到的知识点。本课程将带领大家对SpringAop的实现原理，详细用法和经典场景一步步进行解读，让我们一起探秘Spring Aop！
